## Usage

To get a local copy up and running follow these simple steps.


1. Clone the repo
   ```sh
   git clone https://github.com/raqelcb/push_swap.git
   ```
2. Run `make` (rules: `all, clean, fclean, re`)
   ```sh
   make
   ```
3. Execute
   ```c
   ./push_swap 2 1 3 6 5 8
   ```
